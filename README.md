# WSAI_IITM

Repository containing the work completed during my internship at the **Wadhwani School of Data Science and AI (WSAI), Indian Institute of Technology Madras**.

## Project

**Title:** *An Exploratory Study on AI and Mental Health*

This project explores the use of **Large Language Models (LLMs)** for mental health by integrating Indian personas with counselling conversations to develop persona-aware prompting frameworks and benchmark datasets.

