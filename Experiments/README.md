# Experiments

This directory contains the experiments carried out during my internship at the **Wadhwani School of Data Science and AI (WSAI), IIT Madras**.

The experiments focus on integrating Indian personas with mental health conversations.

---

## Directory Structure

| Folder | Description |
|--------|-------------|
| `1_IndiBias + MentalChat16k` | Pilot experiment combining IndiBias personas with MentalChat16K conversations |
| `2_Nemotron + EmoCare` | Persona-aware prompting using NVIDIA Nemotron Personas and EmoCare |
| `3_Nemotron + EmoCare + User_Type` | Final prompting framework integrating personas, EmoCare conversations, and AI user types |

---

## Experiment Overview

### Experiment 1

**IndiBias + MentalChat16K**

- Semantic matching between personas and conversations
- Persona prefix generation
- Prompt construction
- Analysis of prompt quality

---

### Experiment 2

**Nemotron Personas + EmoCare**

- Selection of representative Indian personas
- Persona-aware prompt generation
- Response generation using LLMs

---

### Experiment 3

**Nemotron Personas + EmoCare + AI User Types**

Final experimental pipeline including:

- Complete Indian persona profiles
- AI user types
- Semantic selection of EmoCare conversations
- Prompt generation
- Response generation using multiple LLMs

---

## Models Used

- Gemini 2.5 Flash
- Gemma 3 27B (via Cerebras)

---

## Objective

The overall objective of these experiments is to explore how detailed Indian personas can improve personalization, cultural awareness, and contextual relevance in AI-generated mental health responses.
