# Datasets

This directory contains all datasets collected, analyzed, and used during my internship at the **Wadhwani School of Data Science and AI (WSAI), IIT Madras**.

The datasets were used for literature review, exploratory data analysis (EDA), persona construction, prompt generation, and evaluation of Large Language Models (LLMs) for mental health applications in the Indian context.

---

## Directory Structure

| Folder | Dataset |
|--------|---------|
| `1_MHQA` | Mental Health Question Answering (MHQA) |
| `2_MH Literacy` | AI-Driven Mental Health Literacy Dataset |
| `3_Dreaddit` | Stress Detection from Reddit Posts |
| `4_SDCNL` | Suicide Detection and Crisis Classification Dataset |
| `5_MentalChat16K` | Mental Health Instruction-Tuning Dataset |
| `6_EmoCare` | Mental Health Counselling Conversations |
| `7_CounselChat` | Professional Counselling Q&A Dataset |
| `8_MentalHealthIndia` | Reddit Dataset collected from r/MentalHealthIndia |
| `9_HOPE` | Counselling Conversation Dataset |
| `10_IndieMH` | Indian Mental Health Instruction Dataset |
| `11_IndiBias` | Indian Persona & Bias Benchmark Dataset |
| `12_Nvidia Nemotron Personas India` | Indian Persona Dataset |

---

## Contents of Each Folder

Each dataset folder may contain:

- Original dataset
- Exploratory Data Analysis (EDA) notebook

---

## Purpose

These datasets support the development of persona-aware mental health AI by:

- Studying existing mental health datasets
- Understanding dataset characteristics through EDA
- Building persona-aware prompts
- Evaluating LLM responses
- Creating benchmark datasets for future research
